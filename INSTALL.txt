KINC
==========
To Install KINC the following dependencies must first be met:

* GNU Scientific Library (http://www.gnu.org/software/gsl/)
* MixModLib (http://www.mixmod.org/article.php3?id_article=55)
* LAPACK (http://www.netlib.org/lapack/)
* BLAS (http://www.netlib.org/blas/)

Before compiling, first edit the Makefile and change the following
variables to suit your environment:

* CC:       set the compiler. Include -m64 flag for compilation on
            64-bit machines
* EXE_DIR:  The location where the kinc binaries should be installed.

To compile simply run

make
make install
